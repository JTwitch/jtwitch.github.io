JTwitch

JTwitch is Twitch api wrapper to create twitch bot with JVM langage.

Use it with maven :

    <dependency>
        <groupId>io.github.jtwitch</groupId>
        <artifactId>jtwitch</artifactId>
        <version>1.0.0</version>
    </dependency>

Or gradle :

implementation 'io.github.jtwitch:jtwitch:1.0.0'
